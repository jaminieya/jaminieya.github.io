# Academic Pages

A GitHub Pages template for academic personal websites. Forked from [academicpages.github.io](https://github.com/academicpages/academicpages.github.io).

## Getting Started

1. Set site-wide configuration in `_config.yml`
2. Add your content to `_pages/about.md`
3. Push to your `username.github.io` repository

## Running locally

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000` to preview.
